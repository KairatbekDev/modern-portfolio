# Фермерский Рынок (Farmer's Market)

## Overview
A React + TypeScript farming market shop UI where users can browse and buy seeds with a virtual wallet.

## Recent Changes
- 2026-02-19: Fixed project setup - added Vite bundler, created index.html, fixed A1.tsx syntax errors, enabled JSX in tsconfig.json

## Project Architecture
- **index.html** - HTML entry point
- **index.tsx** - React app bootstrap (renders App component)
- **A1.tsx** - Main App component with shop UI
- **vite.config.ts** - Vite dev server configuration (port 5000)
- **tsconfig.json** - TypeScript configuration
- **package.json** - Dependencies and scripts

## Running
- `npm run dev` - Start Vite dev server on port 5000
- `npm run build` - Production build
